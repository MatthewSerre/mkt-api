Drivers
GET api/v1/drivers Returns a list of all drivers
GET api/v1/driver/babies Returns a list of all baby drivers
GET api/v1/driver/high_end Returns a list of all High-End drivers
GET api/v1/driver/name?q= Returns the driver with the specified name (e.g. mario, lakitu_(party_time))
